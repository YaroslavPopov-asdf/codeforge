# Ссылки и указатели в C++

## Указатели

```cpp
int x = 42;
int* p = &x;
cout << *p;  // 42
```

## Ссылки

```cpp
int x = 42;
int& r = x;
r = 10;  // теперь x = 10
```

## Передача по ссылке

```cpp
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}
```

Ссылки безопаснее указателей: они не могут быть nullptr и не требуют разыменования.
