# Калькулятор площади прямоугольника

Напишите программу на C++, которая вычисляет площадь прямоугольника.

## Требования

1. Создайте класс `Rectangle` с приватными полями `width` и `height`
2. Конструктор принимает ширину и высоту
3. Метод `area()` возвращает площадь
4. В `main` создайте объект и выведите площадь

## Пример

```
Ввод: 5 3
Вывод: 15
```
