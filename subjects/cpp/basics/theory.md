# Основы C++

C++ — это язык программирования, расширяющий C объектно-ориентированными возможностями.

## Первая программа

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

## Ввод и вывод

```cpp
int x;
cin >> x;
cout << "x = " << x << endl;
```

- `cout` — вывод (console output)
- `cin` — ввод (console input)
- `endl` — конец строки
