# Банковский счёт

Создайте класс `BankAccount` с:
- приватным полем `double balance`
- конструктором, устанавливающим `balance = 0`
- методом `void deposit(double amount)` — пополнение
- методом `bool withdraw(double amount)` — снятие, возвращает false, если недостаточно средств
- методом `double getBalance()`

Программа считывает операции: 0 для выхода, 1 для пополнения, 2 для снятия.

## Пример

Ввод:
```
1
100
2
30
0
```
Вывод:
```
70
```
