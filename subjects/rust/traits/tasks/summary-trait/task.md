# Трейт Summary

Создайте трейт `Summary` с методом `fn summarize(&self) -> String`. Реализуйте его для структуры `Book` с полями `title: String` и `author: String`.

Метод должен возвращать строку вида: `"«{title}» — {author}"`

Программа считывает название и автора и выводит сводку.

## Пример

Ввод:
```
1984
George Orwell
```
Вывод:
```
«1984» — George Orwell
```
