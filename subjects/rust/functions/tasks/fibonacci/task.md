# Числа Фибоначчи

Напишите программу на Rust, которая выводит первые n чисел Фибоначчи.

Ряд Фибоначчи начинается с 0 и 1, каждое следующее число равно сумме двух предыдущих: 0, 1, 1, 2, 3, 5, 8, 13, ...

## Требования

1. Программа читает целое число n из стандартного ввода
2. Выводит первые n чисел Фибоначчи, каждое на новой строке

## Пример

```
Ввод: 6
Вывод: 0
1
1
2
3
5
```
