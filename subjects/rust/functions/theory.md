# Функции и рекурсия в Rust

## Определение функций

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b  // без точки с запятой — это возвращаемое значение
}
```

## Рекурсия

Rust поддерживает рекурсивные функции:

```rust
fn factorial(n: u64) -> u64 {
    if n <= 1 {
        1
    } else {
        n * factorial(n - 1)
    }
}
```

## Замыкания

```rust
let add = |a, b| a + b;
println!("{}", add(3, 5));  // 8
```

## Возврат нескольких значений

```rust
fn div_mod(a: i32, b: i32) -> (i32, i32) {
    (a / b, a % b)
}

let (d, m) = div_mod(10, 3);
```
