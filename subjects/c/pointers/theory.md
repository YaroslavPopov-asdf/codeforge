# Указатели в C

Указатель — это переменная, которая хранит адрес памяти.

## Объявление и использование

```c
int x = 42;
int *ptr = &x;   // ptr хранит адрес x
printf("%d\n", *ptr);  // разыменование — 42
```

## Указатели и функции

Указатели позволяют изменять переменные внутри функций:

```c
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
```

## Указатели и массивы

Имя массива — это указатель на первый элемент:

```c
int arr[] = {1, 2, 3};
int *p = arr;  // то же, что int *p = &arr[0]
printf("%d\n", *(p + 1));  // arr[1]
```
