# Рекурсия в C

Рекурсивная функция — это функция, которая вызывает саму себя.

## Структура рекурсивной функции

```c
int factorial(int n) {
    if (n <= 1) {
        return 1;  // базовый случай
    }
    return n * factorial(n - 1);  // рекурсивный случай
}
```

Каждая рекурсивная функция должна иметь:
1. **Базовый случай** — условие завершения
2. **Рекурсивный случай** — вызов себя с изменёнными аргументами
