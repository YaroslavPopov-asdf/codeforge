# Hello, World!

Напишите программу на C, которая выводит "Hello, World!" на экран.

## Требования

1. Программа должна использовать заголовочный файл `stdio.h`
2. Функция `main` должна возвращать `0`
3. Вывод должен быть точь-в-точь: `Hello, World!`

## Пример

```
Ввод: (нет)
Вывод: Hello, World!
```

## Подсказка

Используйте функцию `printf` из стандартной библиотеки.
