# Переменные и ввод-вывод в C

## Переменные и типы данных

C — статически типизированный язык. Каждая переменная имеет тип:

| Тип | Описание | Пример |
|-----|----------|--------|
| `int` | Целое число | `int x = 5;` |
| `float` | Число с плавающей точкой | `float pi = 3.14;` |
| `double` | Двойная точность | `double d = 3.14159;` |
| `char` | Символ | `char c = 'A';` |

## Ввод и вывод

```c
int age;
printf("Сколько вам лет? ");
scanf("%d", &age);
printf("Вам %d лет\n", age);
```

Спецификаторы формата: `%d` — целое, `%f` — float, `%lf` — double, `%c` — символ.
