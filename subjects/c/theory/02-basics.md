## Переменные и типы данных

| Тип | Размер | Описание |
|-----|--------|----------|
| `int` | 4 байта | Целое число |
| `float` | 4 байта | Число с плавающей точкой |
| `double` | 8 байт | Число с двойной точностью |
| `char` | 1 байт | Символ |

```c
int age = 25;
float pi = 3.14;
char letter = 'A';
```

## Условные операторы

```c
if (x > 0) {
    printf("Положительное");
} else if (x < 0) {
    printf("Отрицательное");
} else {
    printf("Ноль");
}
```

## Циклы

```c
// for loop
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

// while loop
int i = 0;
while (i < 10) {
    printf("%d\n", i);
    i++;
}
```
