# Строки в C

Строки в C — это массивы символов, заканчивающиеся нулевым символом `\0`.

## Объявление

```c
char str[] = "hello";
char str2[100];
```

## Ввод строки

```c
char name[50];
fgets(name, 50, stdin);  // безопасный ввод с пробелами
```

## Функции из string.h

```c
#include <string.h>

strlen(s)     // длина строки
strcpy(dest, src)  // копирование
strcmp(a, b)      // сравнение (0 если равны)
```
