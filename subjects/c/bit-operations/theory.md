# Битовые операции в C

## Основные операторы

- `&` — побитовое И
- `|` — побитовое ИЛИ
- `^` — исключающее ИЛИ (XOR)
- `~` — побитовое НЕ
- `<<` — сдвиг влево
- `>>` — сдвиг вправо

## Примеры

```c
int x = 5;  // 0101
int y = 3;  // 0011
int z = x & y;  // 0001 = 1
int w = x | y;  // 0111 = 7
int s = x << 1; // 1010 = 10 (умножение на 2)
```
