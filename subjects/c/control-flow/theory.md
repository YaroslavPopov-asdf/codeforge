# Условные операторы и циклы

## Условные операторы

```c
if (x > 0) {
    printf("Положительное");
} else if (x < 0) {
    printf("Отрицательное");
} else {
    printf("Ноль");
}
```

Операторы сравнения: `==`, `!=`, `<`, `>`, `<=`, `>=`
Логические операторы: `&&` (И), `||` (ИЛИ), `!` (НЕ)

## Цикл for

```c
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

Три части: инициализация; условие; шаг.

## Цикл while

```c
int i = 0;
while (i < 10) {
    printf("%d\n", i);
    i++;
}
```

## Цикл do-while

```c
int i = 0;
do {
    printf("%d\n", i);
    i++;
} while (i < 10);
```
