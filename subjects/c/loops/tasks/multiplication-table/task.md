# Таблица умножения

Напишите программу, которая выводит таблицу умножения для числа n (от 1 до 9).

Сначала считывается число n, затем выводятся строки вида "n * i = результат".

## Пример

Ввод: 3
Вывод:
```
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9
3 * 4 = 12
3 * 5 = 15
3 * 6 = 18
3 * 7 = 21
3 * 8 = 24
3 * 9 = 27
```
