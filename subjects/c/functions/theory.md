# Функции в C

## Определение функции

```c
int add(int a, int b) {
    return a + b;
}
```

- Тип возвращаемого значения указывается перед именем
- Параметры указываются в скобках с типами
- `return` возвращает значение

## Вызов функции

```c
int result = add(5, 3);
printf("%d\n", result);
```

## Прототипы функций

Если функция определена после main, нужно объявить её прототип:

```c
int add(int a, int b);  // прототип

int main() {
    int sum = add(5, 3);
}

int add(int a, int b) {
    return a + b;
}
```
