# Условные операторы в C

## if-else

```c
if (x > 0) {
    printf("Положительное\n");
} else if (x < 0) {
    printf("Отрицательное\n");
} else {
    printf("Ноль\n");
}
```

## Операторы сравнения

`==` равно, `!=` не равно, `<` меньше, `>` больше, `<=` меньше или равно, `>=` больше или равно.

## Логические операторы

`&&` — И, `||` — ИЛИ, `!` — НЕ.
