# Введение в C

C — это язык программирования, разработанный в 1972 году Деннисом Ритчи. Это один из самых влиятельных языков, на котором написаны операционные системы, встраиваемые системы и множество других критически важных программ.

## Основные характеристики

- **Компилируемый язык** — исходный код преобразуется в машинный код
- **Низкоуровневый доступ** — прямая работа с памятью через указатели
- **Минималистичный** — небольшое количество ключевых слов
- **Быстрый** — высокая производительность исполнения

## Базовая структура программы

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

- `#include <stdio.h>` — подключает стандартную библиотеку ввода-вывода
- `int main()` — точка входа в программу
- `printf()` — функция для вывода текста
- `return 0;` — возвращает код успешного завершения

## Переменные и типы данных

| Тип | Размер | Описание |
|-----|--------|----------|
| `int` | 4 байта | Целое число |
| `float` | 4 байта | Число с плавающей точкой |
| `double` | 8 байт | Число с двойной точностью |
| `char` | 1 байт | Символ |

```c
int age = 25;
float pi = 3.14;
char letter = 'A';
```

## Условные операторы

```c
if (x > 0) {
    printf("Положительное");
} else if (x < 0) {
    printf("Отрицательное");
} else {
    printf("Ноль");
}
```

## Циклы

```c
// for loop
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

// while loop
int i = 0;
while (i < 10) {
    printf("%d\n", i);
    i++;
}
```

## Функции

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    int sum = add(5, 3);
    printf("Сумма: %d\n", sum);
    return 0;
}
```

## Указатели

Указатели хранят адреса памяти. Это одна из ключевых особенностей C.

```c
int x = 42;
int *ptr = &x;  // ptr хранит адрес x
printf("%d\n", *ptr);  // разыменование — получим 42
```

## Массивы

```c
int numbers[5] = {1, 2, 3, 4, 5};
printf("%d\n", numbers[0]);  // первый элемент
```
