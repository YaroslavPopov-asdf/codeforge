# Ваша первая программа на C

Создайте свою первую программу на C. Вы научитесь использовать функцию `printf` для вывода текста.

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

- `#include <stdio.h>` — подключает стандартную библиотеку ввода-вывода
- `int main()` — точка входа в программу
- `printf` — функция для вывода текста
- `\n` — символ новой строки
