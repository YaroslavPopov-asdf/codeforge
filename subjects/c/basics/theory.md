# Основы C

C — это язык программирования, разработанный в 1972 году Деннисом Ритчи.

## Базовая структура программы

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

- `#include <stdio.h>` — подключает стандартную библиотеку ввода-вывода
- `int main()` — точка входа в программу
- `printf()` — функция для вывода текста
- `return 0;` — возвращает код успешного завершения

## Переменные и типы данных

| Тип | Размер | Описание |
|-----|--------|----------|
| `int` | 4 байта | Целое число |
| `float` | 4 байта | Число с плавающей точкой |
| `double` | 8 байт | Число с двойной точностью |
| `char` | 1 байт | Символ |

```c
int age = 25;
float pi = 3.14;
char letter = 'A';
```

## Форматированный ввод и вывод

```c
int x;
printf("Введите число: ");
scanf("%d", &x);
printf("Вы ввели: %d\n", x);
```

- `%d` — целое число
- `%f` — число с плавающей точкой
- `%c` — символ
- `%s` — строка
